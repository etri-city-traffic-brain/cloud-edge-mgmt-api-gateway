# cloud-edge-mgmt-api-gateway(front)

PaaS 개발자를 위한 오픈데이터 API 제공 Portal

## 사전 환경

* nodejs, npm 설치 필요

  * node Version : v12.18.3
  * npm Version : v6.14.6
  
  #### Database 설정 
    * back ip주소 및 port번호를 config_example.js에 입력
    * config_example.js를 config.js로 수정

  ### start project 
        
  #### application 실행 
    * npm install
    * npm start
